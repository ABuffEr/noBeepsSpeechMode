# NoBeepsSpeechMode #
*	 Autor: Alberto Buffolino.
*	 descarga: [versión 1.2][1]

Este complemento excluye los pitidos en el modo de voz cuando se pulsa
NVDA+s.  Sólo puedes conmutar entre desactivado o hablar.
